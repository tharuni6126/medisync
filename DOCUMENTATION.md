# 📘 MediSync Documentation

## Project Name
MediSync

## Project Description
MediSync is a Digital Health Record System developed for the Google Cloud Hack Sprint Hackathon. It provides patients and healthcare professionals with a secure platform to manage medical records digitally.

---

## Objectives

- Reduce paperwork in hospitals
- Securely store patient records
- Enable quick access to medical history
- Improve communication between doctors and patients
- Support cloud-based healthcare solutions

---

## System Modules

### Patient Module
- User Registration
- Secure Login
- View Profile
- Upload Medical Reports
- View Medical History

### Doctor Module
- Login
- View Patient Records
- Update Prescriptions
- Add Medical Notes

### Admin Module
- Manage Users
- Monitor Records
- System Maintenance
